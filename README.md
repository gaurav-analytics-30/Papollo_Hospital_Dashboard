# Papollo_Hospital_Dashboard
📊 Project Overview

This project presents an interactive Healthcare Dashboard built in Power BI to analyze hospital operations, patient flow, and financial performance.

The dashboard helps track patient admissions, discharge trends, follow-ups, billing insights, bed occupancy, and diagnosis patterns to support better decision-making in hospital management.

 Dataset Summary
Time Period: December 2022 – March 2024
Key Fields:
Patient ID
Admit Date
Discharge Date
Follow-up Date
Doctor
Diagnosis
Billing Amount
Bed Occupancy
Health Insurance
